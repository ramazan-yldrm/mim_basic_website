# basic Website Project

A beginner-friendly multi-page website project built using only HTML and CSS.

## Description

This project is a basic website created for learning and practice purposes.  
It focuses on core web development concepts without using JavaScript or external libraries.

The website includes:

- At least five HTML pages
- Column-based layout
- Top navigation menu and sidebar
- Images
- Embedded video and audio
- Tables
- Dark theme styling

## Technologies Used

- HTML5
- CSS3

## How to Run the Project

This project does not require a server or any additional tools.

### Option 1: Open Directly in a Browser

1. Open the `source` folder
2. Double-click `index.html`
3. The website will open in your default web browser

Alternatively:
- Right-click `index.html`
- Select **Open with** → Chrome, Firefox, or Edge

### Option 2: Run Using Visual Studio Code (Recommended)

1. Open the project folder in **Visual Studio Code**
2. In the Explorer panel, open the `source` folder
3. Right-click on `index.html`
4. Select **Open with Live Server**

> If Live Server is not installed:
> - Go to the 'Extensions' tab in VS Code
> - Search for **Live Server**
> - Install it
> - Repeat the steps above

With Live Server:
- The page opens in your browser automatically
- Changes are reflected instantly when you save files
